# EPK page (direct link only)

This page is not linked from the main site and uses `noindex, nofollow` so it is only findable by direct link (e.g. `https://www.crazier.wtf/epk`).

**To add your EPK PDF:** place your file as `epk.pdf` in this folder (`epk/epk.pdf`). It will be embedded on the page and available as a download link.
